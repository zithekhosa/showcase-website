# TOV Technologies Portfolio Showcase

A React + Vite portfolio site for presenting multi-industry web mockups and converting visitors into client inquiries.

## Tech Stack

- React 19
- Vite 7
- Tailwind CSS
- Framer Motion
- React Router

## Local Development

```bash
npm install
npm run dev
```

## Production Checks

```bash
npm run lint
npm run build
```

## Routes

- `/` Home showcase and contact CTA
- `/aura` Wellness mockup
- `/gearhead` Auto parts mockup
- `/guardian` Security services mockup
- `/savor` Restaurant mockup
- `/velvet` Events and weddings mockup
