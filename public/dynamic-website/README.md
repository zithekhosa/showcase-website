# Dynamic Insurance Brokers - Static Mockup Starter

This project is a static website scaffold for mockup and UI exploration.

## Quick start

1. Open `index.html` directly in a browser, or
2. Run a local server (recommended):

```powershell
cd "c:\Users\TOV tech\Desktop\dynamic website"
python -m http.server 5500
```

Then open `http://localhost:5500`.

## Project structure

```text
.
|-- index.html
|-- assets
|   |-- css
|   |   `-- styles.css
|   |-- js
```

## What is included

- Responsive layout shell (header, hero, services, about, CTA, footer)
- CSS design tokens in `:root` for easy theme changes
- Mobile navigation toggle
- Starter content model in `data/company.json` for future data-driven pages
